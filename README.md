# GG Group — Coming Soon

A minimal Tailwind static page to deploy on Vercel.

## Deploy

1. Create a new GitHub repo and add these files.
2. In Vercel, click **New Project** → import the repo.
3. Framework preset: **Other** (static). No build command required.
4. Deploy, then add your domain in Project → **Domains**.
5. Update the email address inside `index.html` if needed.
